O QUE É PROGRAMAR?

  • Consiste basicamente informar o computador de forma de Linguagem de Programação, uma série de passos a ser seguidos de forma ordenada para o funcionamento de determinada coisa.
    Exemplo: Uma receita de bolo precisa ser seguida da forma que está escrita, primeiro colocar os ingredientes em um lugar, misturar, bater, colocar na forma e assar. Basicamente, o programação você dá comandos para a máquina como se fosse uma receita de bolo.

UM POUCO SOBRE O PYTHON

  • Lingugem extremamente versátil,

  • Tipagem dinâmica e forte

    Tipagem Dinâmica: O Python é capas de definir a tipagem de uma variável sem precissar-mos especificar.
    Tipagem Forte: significa que, mesmo que a linguagem permita mudar o tipo de uma variável, ela não deixa você fazer operações incompatíveis com o tipo. Por exemplo, em Python, você não consegue somar uma string com um número sem converter explicitamente o tipo.
    Em resumo: tipagem dinâmica permite mudar o tipo de uma variável durante o programa, e tipagem forte impede operações erradas entre tipos diferentes sem conversão.

  • Multiplataforma (pode ser usada em diversas sistemas, como Windows, Linux, MacOS) e Multiparadigma (suporta diversos paradigmas),

  • Comunidade grande e bem ativa,

  • Curva de aprendizado baixa.

  • Não é boa para app mobile

TIPOS DE DADOS

O Que São Tipos?

Tipos de dados são como "categorias" ou "tipos" para organizar as informações que você usa no seu programa.
Cada tipo de dado define que tipo de valor aquela informação pode ter.

Alguns tipos de dados mais comuns:

  • Número inteiro (int): São números sem decimais, como 1, 20, -5.
  • Número com decimal (float): São números que têm uma parte decimal, como 3.14, -0.5, 100.0.
  • Texto (str): São palavras ou frases, sempre entre aspas, como "olá", "programação", "123".
  • Booleano (bool): Só pode ser verdadeiro (True) ou falso (False). Usado para decisões, como em "Está chovendo?" (sim ou não).
Esses tipos ajudam o computador a saber como tratar as informações.
Por exemplo: Se você tentar somar um número com uma palavra, o computador vai ter dificuldades, porque são tipos diferentes.

MODO INTERATIVO

  • O interpretador do Python pode ser acessado para o programador escrever um código e mostrar o resultado na hora.
  • Para iniciar o modo interativo, basta ir no terminal e digitar apenas "python" ou executando o script com a flag -i, como (python-i app.py), colocando o arquivo ,py no final
  • Para sair do modo interativo, aperte (Ctrl+z)

CONSTANTES E VARIÁVEIS

VARIÁVEIS:

  • São basicamente valores que podem sofrer alterações no decorrer da execução do programa.
    São variáveis, por nascerem com um valor e não necessariamente continuar com ele até o final da execução do programa.
  • Pode declarar várias variáveis de uma vez, basta apenas colocar ",".
    Exemplo: nome, idade = "Leandro", 21.

    • CONSTANTES:

  • Assim como as variáveis, as constantes também são utilizadas para armazenar valores, a diferença é que a constante mantém o valor até o final da execução do programa.

    • NO PYTHON NÃO TEMOS CONSTANTES:
    Na linguagem Python, não temos uma palavra para constante como em outras linguagens, como no Java e C que utiliza-se de final e const para declarar uma constante
    Em Python, utilizamos a convenção que diz ao programador que uma variável é uma constante. Para fazer isso, você precisa criar uma variável totalmente em letras maiúsculas.

    BOAS PRÁTICAS

    • O padrão de nomes de variáveis e constantes deve ser em snake case (substitui espaços por underlines "_")
      Exemplo: salario_minimo = 1300.00,
    • Escolher nomes sugestivos,
    • Nome de constantes todo em maiúsulo.
